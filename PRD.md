# 产品需求文档 (PRD)

## 产品名称

Yixiao 华语学习工具

## 用户与使用场景

- **主用户**：Yixiao，新加坡小学四年级，学母语华语（Higher Chinese / 华文）
- **使用习惯**：每天读华文文章，在纸上划线/圈词/写旁批，学习后录入系统
- **设备**：手机（拍照录入）+ 电脑（浏览、复习），两端数据实时同步
- **语言背景**：英文是主要语言，华文是第二语言 → 字词解释**全部用中文**，不加英文辅助

**当前阶段**：单用户（Yixiao），无需登录界面  
**将来扩展**：数据架构支持多用户，可加妹妹及其他家庭

---

## 功能模块

### 模块 1：文章录入

#### 录入方式 A：拍照上传（优先级最高）

用手机拍下书页，系统**全自动**处理，不需要 Yixiao 手动调整：

| 处理层次 | 具体能力 |
|----------|----------|
| **L4 图像预处理** | 自动纠偏歪斜纸张、裁掉背景杂物、提升曝光不足的亮度、识别并处理轻微运动模糊 |
| **L3 版面理解** | 自动识别单栏/双栏排版、跨页拍摄（两页内容同时入镜）、只拍了半页的情况，正确提取正文 |
| **L2 批注识别** | 识别 Yixiao 在纸上划的下划线（标记生词）、圈出的词语、旁边写的手写笔记，单独提取 |
| **L1 OCR** | 提取书页上的汉字，还原为可编辑文本 |

**录入完成后展示**：
- 文章正文（可在界面上阅读）
- 被划线/圈出的词高亮显示
- 手写旁批转成文字附在旁边
- 原始照片可查看

#### 录入方式 B：内置教材

内置新加坡 MOE 小学华文词汇表（约 2000 词，分级）作为字典底库。  
课文原文因版权暂不内置，实际课文内容通过拍照录入。

---

### 模块 2：字词查询

**触发方式**：在文章界面点击任意汉字或词语，弹出字词卡片。

**卡片内容**：

| 字段 | 说明 |
|------|------|
| 拼音 | 按新加坡 MOE 标准标注 |
| 词性 | 名词 / 动词 / 形容词 / 副词等 |
| 释义 | 用小学生能懂的中文表述，简洁清晰 |
| 例句 | 2-3 个，小学课文程度 |
| 用法说明 | 常见搭配、使用场合 |
| 关联词 | 见模块 3 |

**性能**：同一个词的查询结果缓存到数据库，不重复调用 AI。

---

### 模块 3：知识关联

**目标**：把不同时间、不同文章里学到的词自动联系起来。

**关联类型**：

| 类型 | 例子 |
|------|------|
| 同字词 | 柔软 → 柔弱、柔和、柔顺 |
| 近义词 | 茂盛 → 繁茂、葱郁 |
| 反义词 | 茂盛 ↔ 稀疏 |
| 相关词 | 秋天 → 落叶、丰收、凉爽 |

**推荐范围**（两种都显示，视觉区分）：
- **已学过的词**：Yixiao 词库里有的，直接跳转到她的学习记录
- **尚未学过的相关词**：标注为"扩展词"，供延伸学习

**触发时机**：保存新词时系统自动在后台计算关联，不影响当前操作。

**展示方式**：
- 字词卡片底部显示关联词列表（已学词 + 扩展词视觉区分）
- 点击可跳转到对应词的学习记录
- 第二期考虑可视化关系图谱

---

### 模块 4：学习档案与复习

#### 学习档案

- 文章时间线（按阅读日期排列，可点击进入全文）
- 词汇表：所有学过的词，可按文章/日期/掌握状态筛选
- 每个词记录"第一次见到"的文章和日期

#### 复习功能

- **闪卡复习**：展示汉字 → 点击翻面看拼音和释义 → 标记"会了"或"需要再练"
- **复习提醒**：超过 7 天未复习的词，在 App 内提示（仅 App 内，不发手机通知）
- 复习计划基于艾宾浩斯遗忘曲线（1天、3天、7天、15天、30天复习间隔）

---

## 明确不做的功能

- 用户注册 / 登录界面（数据架构预留，但当前不实现）
- 写作练习、造句练习等主动输出功能
- 听力练习、跟读录音功能
- 社交功能（分享、比赛等）

---

## 验收标准

| 场景 | 通过标准 |
|------|----------|
| 拍一页歪斜的华文书页 | 自动纠偏，正文被正确提取 |
| 书页上有下划线 | 被划线的词自动高亮，进入学习词库 |
| 点击"茂盛" | 显示拼音、释义、例句，关联词里出现"稀疏" |
| 先学"柔软"，再学"柔弱" | 两词互相出现在对方的关联词列表里 |
| 手机录入，电脑打开 | 数据实时同步，看到相同内容 |
| 7天未复习某词 | App 首页出现复习提醒 |
